# xo_in_console

xo game on terminal

## Installation

Install xo_in_console with git clone

```bash
  git clone https://github.com/DevMykola/xo_in_console.git
  cd xo_in_console
  git clone https://github.com/DevMykola/Games.git
```
    
## Run Locally

Go to the project directory

```bash
  cd xo_in_console
```

Start the game

```bash
  python3 main.py
```
